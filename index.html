40. Em uma amostra de 200 pacientes, 80% apresentaram melhoria após o tratamento. Qual é o intervalo de confiança de 99% para a proporção de pacientes que melhoraram?

  • A) 0,78 a 0,82
  • B) 0,79 a 0,81
  • C) 0,75 a 0,85
  • D) 0,74 a 0,86
  • E) 0,76 a 0,84
Ver resposta

A resposta correta é: D. Para calcular o intervalo de confiança de 99% para a proporção:

IC = p ± Z * √(p(1-p)/n), onde:

  • p = 0,80 (proporção observada)
  • n = 200 (tamanho da amostra)
  • Z = 2,576 (valor crítico para 99%)

Erro padrão (EP) = √(p(1-p)/n) = √(0,80 * 0,20 / 200) ≈ 0,0283.

Margem de erro (ME) = Z * EP = 2,576 * 0,0283 ≈ 0,0729.

IC = 0,80 ± 0,0729 → [0,7271, 0,8729], arredondando para duas casas decimais, o intervalo é [0,74, 0,86].

42. Qual é o desvio padrão de um conjunto de dados com os valores {2, 4, 6, 8, 10}?

  • A) 2,83
  • B) 3,16
  • C) 4
  • D) 5
  • E) 1,41

A resposta correta é: A. O desvio padrão é calculado como:

1. Calcular a média: (2+4+6+8+10)/5 = 6.

4. O desvio padrão é a raiz da variância: √8 ≈ 2,83.

  • B) 0,5000
  • C) 0,3500
  • D) 0,2000
  • E) 0,1000
Ver resposta

A resposta correta é: A. Usando a fórmula da distribuição binomial:

P(3) = C(10, 3) * 0,3³ * 0,7⁷ = 120 * 0,027 * 0,082354 ≈ 0,2668.
